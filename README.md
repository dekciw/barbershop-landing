# Барбершоп — лендинг сайта

Веб‑сайт барбершопа: адаптивный лендинг с секциями «О нас», услуги, мастера, отзывы и контакты. Есть форма записи с маской телефона и датапикером, слайдеры на Slick и анимации появления блоков.

— Репозиторий: `dekciw/1.barbershop-website`

## Краткое описание

Проект создан для практики вёрстки и фронтенда: работа со сборкой стилей (SCSS → CSS), подключение сторонних JS‑плагинов, настройка интерактива (формы, выпадающие списки, карусели) и базовой анимации на лендинге.

## Основные возможности

- **Адаптивная вёрстка**: корректное отображение на десктопах и мобильных устройствах.
- **Секции лендинга**: герой‑экран, о компании, услуги, мастера, отзывы, контакты.
- **Форма записи**: маска телефона и выбор даты (Air‑Datepicker), валидация на стороне клиента.
- **Слайдеры**: карусели на базе Slick (галереи/отзывы/мастера).
- **Анимации**: WOW.js для появления блоков при прокрутке.

## Технологии

- **HTML5**, **SCSS/CSS3**, **JavaScript** (jQuery для плагинов)
- Плагины: `Slick`, `Air‑Datepicker`, `jquery.maskedinput`, `WOW.js`
- Сборка стилей через `sass`/`gulp` и минификация CSS (готовая сборка в `dist/style.min.css`)

## Как это работает

1. На странице подключаются стили из `dist/style.min.css` и скрипты из `src/scripts`.
2. `src/scripts/index.js` инициализирует плагины:
   - Slick для слайдеров (на блоках «Мастера», «Отзывы» и т. п.).
   - Air‑Datepicker на поле даты в форме записи.
   - Маску телефона через `jquery.maskedinput`.
   - WOW.js для анимаций при скролле.
3. SCSS собирается и минифицируется в один файл CSS, подключённый в `index.html`.

## Цели проекта

- Практика модульной вёрстки и организации стилей (SCSS, разбивка по файлам).
- Подключение и конфигурация популярных фронтенд‑плагинов (Slick, Air‑Datepicker, WOW.js).
- Работа с формами: маски, валидация, UX мелочи (подсказки/плейсхолдеры).
- Подготовка продакшен‑стилей (минификация и структура `dist/`).

## Локальный запуск

Проект статический — можно открыть `index.html` напрямую в браузере. Для работы со SCSS и минификацией стилей:

```bash
npm install
npx gulp  # или скрипты из gulpfile.js для сборки/минификации
```
